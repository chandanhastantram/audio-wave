# Audio Wave

A music player with real-time audio visualizer.

## Features

- Audio playback controls (play, pause, next, previous)
- Real-time frequency visualizer
- Volume control
- Seek functionality
- Playlist management
- Black and blue/green theme

## Tech Stack

- React
- Vite
- Web Audio API
- Canvas API

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deploy the `dist` folder to any static hosting service.
